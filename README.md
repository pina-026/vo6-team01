### 使い方

```sh
$ make # docker-compose up -d をする

$ make java/bash # Java コンテナに入る
$ make react/ash # React コンテナに入る

$ make mysql # MySQLコンテナに入る
$ make redis/bash # Redisコンテナに入る

# 困ったら
$ make help # コマンド一覧が出力される
```
